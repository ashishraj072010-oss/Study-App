# Smart Study Manager

A full-stack study planner web application.

## Integration Notes
- Google Sheets: Integration was skipped during initial setup because authorization was dismissed. The UI includes a placeholder button for connection logic. If this needs to be implemented later, Replit integrations must be authorized or API credentials provided.

## Stack
- Frontend: React, Wouter, Shadcn, Tailwind CSS, Recharts
- Backend: Node.js, Express, Drizzle ORM, PostgreSQL
